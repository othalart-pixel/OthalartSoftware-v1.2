# Proyectos Othalart

Los proyectos guardados automáticamente se organizan en una carpeta propia:

```text
Proyectos/
  Nombre del proyecto/
    Nombre del proyecto.othalart.json
    Bibliotecas/
      cargos.json
      ecuaciones_productivas.json
      etapas.json
      gestiones_productivas.json
      productos2d.json
      rendimientos_productivos.json
      resoluciones_dependencias.json
      subetapas.json
```

El archivo `.othalart.json` continúa siendo autocontenido. La carpeta
`Bibliotecas` conserva además copias separadas y legibles de las bibliotecas
utilizadas al guardar ese proyecto.
